SEO операции которые обязательны при создании сайта:
	Формирование тегов <meta name="description" content="Мета-описание" /> для каждой страницы сайта	подробнее: https://support.google.com/webmasters/answer/35624#1
		- Мета-тег Description должен быть заполнен на всех страницах сайта, которые вы желаете видеть в поисковой выдаче. Иначе в качестве содержимого сниппета будут отображаться непредсказуемые фрагменты основного контента веб-страницы, что может негативно отразиться на кликабельности сниппета 
		- Мета-описание должно быть уникальным для каждой страницы сайта
		- Мета-описание должно описывать конкретную страницу сайта, а не сайт в целом
		- Длина мета-тега Description в 100-140 символов гарантирует отображение его содержимого в сниппете (в случае генерации аннотации сниппета из мета-описания)
		- Если в описании нет ключевых слов – аннотация сниппета, скорее всего, НЕ БУДЕТ формироваться из мета-описания. НО 
		- Метаописания, которые состоят из длинных цепочек ключевых слов, не дают пользователям четкого представления о содержании страницы и вряд ли будут использоваться вместо обычного фрагмента с описанием
		- Текст в мета-тегах Description должен быть ориентирован на потенциальных посетителей сайта из поисковой выдачи
			Мета-описания должны быть написаны для людей, нормальным человеческим языком — развернуто, правильно выстроенными предложениями, без злоупотреблений ключевыми словами, фразами, заглавными буквами, рекламными лозунгами и пр.
		мета-тег Description может содержать спецсимволы Unicode (например: ©, ®, ♥)
